# CLAUDE SONNET 4.6 — BUILD A PROCEDURAL PLANET SURVIVAL GAME

Build a procedural space exploration game called:

"Stellar Frontier"

Inspired by:
- No Man’s Sky
- Outer Wilds
- Elite Dangerous

CORE EXPERIENCE:
Players explore infinite procedural planets seamlessly from orbit to ground.

TECH STACK:
- Three.js
- WebGPU optional
- Procedural terrain shaders

CORE FEATURES:

Generate:
- Mountains
- Oceans
- Alien forests
- Biomes
- Weather systems

2. SEAMLESS SPACE-TO-GROUND TRANSITION
NO loading screens.

Players can:
- Fly into atmosphere
- Land anywhere
- Explore surfaces

3. SURVIVAL SYSTEMS
- Oxygen
- Temperature
- Hunger
- Radiation
- Base building

4. RESOURCE ECONOMY
- Mining
- Crafting
- Fuel systems
- Ship upgrades

5. ADVANCED RENDERING
- Atmospheric scattering
- Planet LOD
- Terrain tessellation
- Dynamic clouds

6. AI ECOSYSTEMS
- Alien creatures
- Dynamic behaviors
- Procedural species

7. MULTIPLAYER
Optional co-op exploration.

PORTFOLIO VALUE:
Demonstrates:
- Procedural generation
- Terrain systems
- Shader engineering
- Optimization
- Streaming worlds
